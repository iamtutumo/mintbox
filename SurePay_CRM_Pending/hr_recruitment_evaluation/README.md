# HR Recruitment - Evaluation System

## Overview
Multi-reviewer scoring and interview management system for recruitment.

## Features
- Multi-reviewer evaluations (Department Head, HR, CEO, Panel)
- Scoring system (Technical, Cultural Fit, Communication)
- Weighted score calculation
- Recommendation tracking
- Interview notes and feedback
- Evaluation wizard for quick entry
- Evaluation summary on applicant

## Installation
Requires: `hr_recruitment`, `hr_applicant_tracking_random`

## Usage
1. Open applicant record
2. Click "Evaluate" button
3. Fill in scores and feedback
4. Submit evaluation
5. View all evaluations in "Evaluations" tab
