# HR Recruitment - Analytics Dashboard

## Overview
Recruitment metrics and analytics dashboard with graphs, pivot tables, and reports.

## Features
- Recruitment analysis reporting model
- Graph views (bar, line, pie charts)
- Pivot tables for data analysis
- PDF reports
- Key metrics: applicants, hires, rejections, conversion rates, time-to-hire
- Filtering by job, department, date range
- Grouping by job, department, stage, month

## Metrics
- Total Applicants
- Hired Count
- Rejected Count
- Average Time to Hire (days)
- Conversion Rate (%)

## Views
- Graph view (bar charts)
- Pivot table (cross-analysis)
- PDF report export
