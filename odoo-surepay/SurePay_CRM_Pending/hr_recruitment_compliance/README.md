# HR Recruitment - Compliance Tools

## Overview
GDPR compliance and audit trail for recruitment data management.

## Features
- GDPR consent tracking
- Data retention management (default 2 years)
- Right to Access (data export in JSON/Text)
- Right to be Forgotten (data anonymization)
- Complete audit trail
- Automated data retention cleanup
- Audit logging for all actions

## GDPR Rights Implemented
1. **Right to Access**: Export all applicant data
2. **Right to be Forgotten**: Anonymize personal data
3. **Right to Data Portability**: Export in JSON format
4. **Consent Management**: Track consent date and source

## Audit Log
Tracks all actions:
- Created
- Viewed
- Updated
- Deleted
- Data Exported
- Anonymized
- Email Sent

## Scheduled Actions
- Daily check for expired data retention
- Auto-anonymization of expired records
